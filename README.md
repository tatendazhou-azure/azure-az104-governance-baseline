## Professional Context
This lab simulates how an Azure Administrator establishes governance controls in a real-world environment to ensure security, cost management, and operational consistency before deploying workloads.

## Overview
This project demonstrates a governance-first approach to Azure administration, focusing on cost control, access management, tagging standards, and policy enforcement.

## Objectives
- Establish a dedicated lab subscription
- Apply role-based access control (RBAC)
- Enforce governance using Azure Policy
- Implement resource tagging standards
- Protect critical resources using locks

## Azure Services Used
- Azure Subscriptions
- Azure Resource Groups
- Azure RBAC (IAM)
- Azure Policy
- Resource Locks
- Cost Management (Budgets)
